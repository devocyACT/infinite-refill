# 归档脚本

本目录包含项目早期的 Shell 脚本实现，已被 Go 版本替代。

## 文件说明

### menu.sh
交互式菜单脚本，提供以下功能：
- 立即单次续杯
- 设置配置
- 打开/关闭定时任务
- 同步所有账号
- 清理失效账号
- 查看账号状态
- 查看日志
- 更新脚本
- 卸载

### refill.sh
核心续杯脚本，实现：
- 账号探测
- Topup 请求
- 增量闭环逻辑
- 定时任务支持

## 迁移说明

这些脚本已被 Go 版本完全替代，提供更好的性能和可维护性：

| 特性 | Shell 版本 | Go 版本 |
|------|-----------|---------|
| 性能 | 串行执行 | 并发执行（6x） |
| 依赖 | jq/curl/python3 | 零依赖 |
| 跨平台 | macOS/Linux | 全平台 |
| 类型安全 | 无 | 编译时检查 |
| 测试 | 无 | 单元测试 |

详见 [MIGRATION_GUIDE.md](../MIGRATION_GUIDE.md)

## 保留原因

保留这些脚本用于：
1. 参考原始实现逻辑
2. 对比性能改进
3. 历史记录和学习

**注意：** 不建议在生产环境使用这些脚本，请使用 Go 版本。
